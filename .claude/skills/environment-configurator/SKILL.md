# Environment Configurator Skill

## Purpose

This skill should be used when setting up comprehensive environment configuration with secrets management, configuration validation, and environment-specific optimizations for the Phase II Todo Full-Stack Web Application. The skill focuses on creating secure, maintainable, and efficient environment management systems.

Use this skill when:
- Setting up environment variables and configuration
- Implementing secrets management and security
- Creating environment-specific optimizations

- Managing configuration validation and error checking
