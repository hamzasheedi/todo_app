# Skill Creator

This skill provides guidance for creating effective skills for Claude. It helps users understand how to structure, develop, and package skills that extend Claude's capabilities with specialized knowledge, workflows, and tool integrations.

## What This Skill Does

The skill-creator skill guides users through the complete skill creation process:

1. **Understanding** - Helps clarify the purpose and use cases for a new skill
2. **Planning** - Assists in planning reusable components (scripts, references, assets)
3. **Initializing** - Provides tools to create the proper directory structure
4. **Editing** - Guides the creation of proper SKILL.md documentation
5. **Packaging** - Validates and packages skills for distribution

## Key Components

- **SKILL.md template** - Provides the structure for documenting new skills
- **init_skill.py** - Script to initialize a new skill with proper directory structure
- **package_skill.py** - Script to validate and package skills for distribution

## Usage

Use this skill when you want to create a new skill or update an existing one that extends Claude's capabilities with specialized knowledge, workflows, or tool integrations.
