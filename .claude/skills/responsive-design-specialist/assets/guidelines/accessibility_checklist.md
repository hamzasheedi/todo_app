# Accessibility Checklist (WCAG 2.1 AA)

## Perceivable

### Text Alternatives
- [ ] All images have appropriate alt text
- [ ] Decorative images have empty alt attributes
- [ ] Complex images have detailed descriptions
- [ ] Captions provided for audio content

### Time-based Media
- [ ] Audio-only content has text alternatives
- [ ] Video content has captions
- [ ] Audio descriptions provided for video content

### Adaptable
- [ ] Content can be presented in different ways
- [ ] Information not conveyed by color alone
- [ ] Headings and labels describe topic or purpose

### Distinguishable
- [ ] Color contrast ratio 4.5:1 for normal text, 3:1 for large text
- [ ] Visual focus indicators are visible
- [ ] Text can be resized up to 200% without assistive technology

## Operable

### Keyboard Accessible
- [ ] All functionality available via keyboard
- [ ] No keyboard traps
- [ ] Skip links provided for main navigation

### Enough Time
- [ ] Timing adjustable for time limits
- [ ] No content flashes more than 3 times per second

### Seizures and Physical Reactions
- [ ] No content that could cause seizures

### Navigable
- [ ] Multiple ways to locate content
- [ ] Headings and labels describe topic or purpose
- [ ] Focus order is logical

## Understandable

### Readable
- [ ] Language of page identified
- [ ] Unusual words defined
- [ ] Abbreviations expanded

### Predictable
- [ ] Changes of context only on user request
- [ ] Consistent navigation mechanisms
- [ ] Consistent identification of functions

### Input Assistance
- [ ] Error prevention for legal, financial, data
- [ ] Labels or instructions for user input
- [ ] Error identification and suggestions for correction

## Robust

### Compatible
- [ ] Compatible with assistive technologies
- [ ] Valid HTML markup
- [ ] Proper ARIA attributes used appropriately
